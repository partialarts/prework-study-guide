Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The header, main and footer elements represent the various parts of the page
  • The section element defines a section in a document that contains thematic content.
  • The h1, h2, h3, h4, h5 and h6 elements represent heading hierarchy - h1 should only be used once on each page
  • The img element represents an image - the src attribute defines the source of the image and the alt tag provides text for accessibility and SEO purposes
  • The id tag is used to reference elements in other parts of the code
  • The ul elements defines an unordered list
  • The li element defines individual list items
  • The p element represents a paragraph or a block of text
  • The a element stands for anchor and creates a link, the href attribute points to the URL for the link

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A * selector assigns the property to the entire document
  • We can style HTML elements using the elements themselves as selectors or by using the class attribute as a selector.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector
  • A class selector is designated with a period, e.g. .card
  • CSS files must be linked to the HTML file in the header using the < link > element
